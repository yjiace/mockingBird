GET /users

检索系统中的用户列表。
此端点用于获取所有已注册用户的列表，可以支持分页和基本筛选。

参数

参数 类型 描述
pagequery integer 请求的页码，用于分页。(默认值: 1) 例如: 1
limitquery integer 每页返回的用户数量。(默认值: 10) 例如: 10
sortByquery string 用于排序的字段名称。例如: "创建时间"

请求示例

HTTP
SSE
GET /users?page=1&limit=10 HTTP/1.1
Host: YOUR_API_BASE_URL
User-Agent: Your-App/1.0
Accept: application/json
